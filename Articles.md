# 웹 폰트
서비스에서 사용하고 있는 웹 폰트가 1MB 정도의 크기라 줄일 수 있는 방법에 대해 알아보았다. `subset` 파일을 사용하면 300KB로 줄어들긴 하지만 퀄리티에 차이가 있지 않을까 싶었다.
